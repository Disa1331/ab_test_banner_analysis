# Анализ эффективности рекламных баннеров и план A/B - тестирования

**Период анализа:** Сентябрь 2025  
**Роль:** Data Analyst  
**Инструменты:** Excel, Python (pandas, numpy, matplotlib), SQL, Яндекс.Метрика

## Цель проекта  
Оценить эффективность рекламных баннеров мобильного приложения по доставке еды и предложить A/B-тестирование для увеличения количества установок.

## Задачи
- Разработать набор метрик для оценки баннеров
- Определить ключевые зависимости и воронку действий пользователей
- Провести визуальный анализ баннерв и сформулировать гипотезы
- Разработать план A/B-тестирования

## Мои действия
- Определил метрики для анализа: Просмотры, Клики, CTR, Установки, CR
- Рассчитал зависимости и воронку: Просмотры ➡︎ Клики ➡︎ Установки
- Провел визуальный анализ баннеров и предложил улучшения:
    - Замена изображения (разнообразие блюд)
    - Яркие цвета и контрасная CTA-кнопка
    - Добавления УТП (Уникального Торгового Предложения) и скидочного бейджа
- Сформулировал гипотезы для A/B-теста:
    - H1: Замена изображения роллов на ассортимент блюд увеличит CTR
    - H2: Яркий акцентный цвет фона и кнопки увеличит CTR
    - H3: Добавление скидочного бейджа повысит CR
- Составил детальный план эксперимента: метрики, выборка, статистическая проверка, длительность

## Ключевые выводы
- Текущие баннеры низкоэффективны: CTR и CR могли быть выше
- Основные точки улучшения: креатив, контраст, слоган, УТП
- Предложенные гипотезы направлены на рост CTR и CR
- Разработан подробный план A/B-тестирования

## Результат
- Сформирована продуктовая стратегия улучшения баннеров
- Определена структура метрик и воронки для оценки рекламных компаний
- Подготовлен пошаговый план проведения A/B-теста
